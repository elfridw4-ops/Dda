# Restaurant - Site Vitrine

## Description
Ce projet est un **site vitrine pour un restaurant**, développé en HTML et CSS. Il permet d'afficher les plats proposés, une section "À propos" et une barre de navigation pour naviguer entre les différentes sections.

### Fonctionnalités
- **Barre de navigation** : Permet de naviguer entre les sections du site.
- **Section "Nos Plats"** : Affiche les plats disponibles avec leur image, description et prix.
- **Section "À propos"** : Présente le chef cuisinier et fondateur du restaurant.
- **Responsivité** : Le site s'adapte aux mobiles et tablettes.
- **Accessibilité** : Utilisation d'attributs `aria-label` et de balises sémantiques pour améliorer l'accessibilité.

## Structure du projet
```
/ecole
│── index.html          # Page d'accueil du site
│── btn.html            # Composant bouton
│── card.html           # Composant carte de plat
│── img.html            # Effet de survol sur une image
│── nav.html            # Barre de navigation
│── profil.html         # Carte de profil utilisateur
│── resto.html          # Détails d'un plat
│── style.css           # Styles centralisés
│── README.md           # Documentation du projet
│
└── /images            # Dossier pour les images (à créer)
```

## Prérequis
- Un navigateur web (Chrome, Firefox, Edge, etc.).
- Un éditeur de code (VS Code, Sublime Text, etc.) pour modifier les fichiers.

## Installation
1. Clonez ce dépôt ou téléchargez les fichiers.
2. Ouvrez le dossier `ecole` dans votre éditeur de code.
3. Ouvrez le fichier `index.html` dans un navigateur pour visualiser le site.

## Utilisation
- **Naviguez** entre les sections via la barre de navigation.
- **Découvrez** les plats proposés dans la section "Nos Plats".
- **Contactez** le chef via le bouton "Contactez-moi" dans la section "À propos".

## Améliorations possibles
- Ajouter un formulaire de contact.
- Intégrer un système de réservation en ligne.
- Ajouter des animations CSS pour améliorer l'expérience utilisateur.
- Utiliser JavaScript pour rendre le site dynamique (ex: filtre des plats).
